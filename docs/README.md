# Low Level GPU Tools for JavaScript Documentation

This directory contains documents for CUDA Fatbin, HIP Fatbin, Microsoft Portable Executable, GNU ELF, SPIR-V, LLVM IR/BC and internal NVIDIA and GPUOpen data and file formats, as well as information on SASS, PTX, AMD GCN, RDNA, CDNA and CUDA C(++).
